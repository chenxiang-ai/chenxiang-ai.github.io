二分查找

作出限制

  • 左右边界必须是合法取值

  • 左右边界时刻保持相反状态（举例：左边界总满足arr[left]<taget，右边界总满足arr[right]>=taget）

  • 循环结束是左右边界相邻

  • 返回结果是左边界还是右边界根据实际题目判断

    • 返回false，表示arr[index]不满足check要求

堆

  • 大顶堆，父结点大于子结点
  • 小顶堆，子结点大于父节点

大根堆（Max Heap）是一种特殊的二叉堆，其中每个节点的值都大于或等于其子节点的值。大根堆有许多用途，包括但不限于以下几个方面：

  1. 优先队列（Priority Queue）：大根堆常常用于实现优先队列。在优先队列中，元素按照优先级顺序进行排列，大根堆能够确保队列中优先级最高的元素始终处于堆顶位置。这样可以在O(1)时间内获取到优先级最高的元素，并且在O(log n)时间内插入新元素或删除堆顶元素。

  2. 堆排序（Heap Sort）：堆排序是一种高效的排序算法，它利用了大根堆的性质。在堆排序中，首先将待排序的元素构建成一个大根堆，然后不断地将堆顶元素与堆的最后一个元素交换，并将堆的大小减一，再对剩余的元素进行堆调整，直到整个序列有序。

  3. 中位数查找：大根堆可以用于在一组动态数据中快速查找中位数。通过维护两个堆，一个大根堆和一个小根堆，可以使得堆顶元素分别为数据序列的中位数或者接近中位数的值。

  4. 滑动窗口问题：在滑动窗口问题中，大根堆可以帮助我们在一个固定大小的窗口内，高效地查找窗口内的最大值或者最小值。

  5. 最大堆化（Max Heapify）：在数据结构和算法中，大根堆可以被用于实现一些其他算法，比如构建哈夫曼树、Dijkstra算法中的优先队列等。

总的来说，大根堆作为一种重要的数据结构，具有广泛的应用场景，在优先队列、排序算法以及一些特定问题的解决中发挥着重要作用。

建堆方法

  • 对于数组中下标为$i$的元素，其子结点的下标为$(i+1)*2$与$(i+1)*2+1$

    • 假设堆结点编号从0开始，构成的树结构的层数从1开始
    • 第n层最后一个结点的编号为$2^n-2(n=0,1,\dots)$,可通过等比数列证明
    • 给定结点编号$i$,其在$n$层，其上一层最后一个结点的编号为$2^{n-1}-2$,$i$与上一层最后一个结点相差$i-2^{n-1}+2$
    • 则$i$的右孩子编号为$2^n-2+2(i-2^{n-1}+2)$，为$2i+2$，左结点为$2i+1$

  • 一个堆中最后一个非叶子结点的编号为$\lfloor n / 2\rfloor - 1$,编号从$0$开始，$n$为数组长度

    • 结合数组中下标为$i$的元素，其子结点的下标为$(i+1)*2$与$(i+1)*2+1$
    • 假设最后非叶子结点编号为$i$，那么最后一个结点编号为$n-1$
      • $n$为奇数，则最后结点为右结点，则有$n-1=2i+2$，$i=\frac{n-1}{2}-1$
      • $n$为偶数，则最后结点为左结点，则有$n-1=2i+1$，$i=\frac{n}{2}-1$
    • 将上述两种情况统一，有$i=\lfloor n / 2\rfloor - 1$

基于上述两个结论，我们从结点$\lfloor n / 2\rfloor - 1$自底向上遍历，将父、子结点中最大的交换至父结点上即可

代码模板

  1. 寻找最长

    • L、R起点为0
    • 判断窗口是否符合要求
      • 符合：更新最优解，移动右指针
      • 不符合：移动左指针直到符合要求或L==R

  2. 寻找最短

    • L、R起点为0
    • 判断窗口是否符合要求
      • 符合：移动左指针并更新最优解，直到不符合要求或L==R
      • 不符合：移动右指针

最长模板通常步骤如下：

  • 将当前右边界元素加入窗口
  • 新窗口不符合条件，内层循环扩展左边界直到窗口符合条件
  • 更新最优解，将右边界扩展至下一个待加入窗口的元素
